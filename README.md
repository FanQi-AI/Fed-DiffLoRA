# Fed-DiffLoRA: Personalized Federated Style Transfer for T2I Diffusion Models

This repository provides the implementation of **Fed-DiffLoRA**, a framework for **federated style transfer in text-to-image diffusion models**. The code separates **style LoRAs** (public, sharable) and **content LoRAs** (private, client-side), and fuses them via a federated aggregation mechanism. This enables **privacy-preserving personalized style transfer** and **high-quality image generation** across distributed clients. The code is currently being organized and will be uploaded soon.
